git

1. 프로젝트 폴더를 루트로 열고 깃으로 초기화

  ctrl + J
  git init (최초1회)

2. 파일을 stage 로 올림

  git add 파일명 (파일명의 파일만 stage 에 올림)
  git add * (모든 파일을 stage 에 올림)
  git add . (변경이력이 있는 파일만 stage 에 올림)

3. 버전을 생성함

git commit -m "최초커밋"

사용자등록 : 최초1회
